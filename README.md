## 简介

这是一个 iOS 项目，已上架 App Store。

本项目的代码**没有一行由本人手写**，全部由 AI 生成完成。生成与移植过程中主要参考了以下仓库：

- Android 端：https://github.com/BIT101-dev/BIT101-Android
